# Automação de Planilhas com Python

Projeto em Python para automatizar o processamento de dados em planilhas Excel.

## Funcionalidades

- leitura de planilha Excel
- cálculo de faturamento por item
- geração automática de relatório

## Tecnologias

- Python
- Pandas
- Openpyxl
